Projects

Spring REST API template

JAVA Spring SpringMail SpringSecurity JWT MySQL Hibernate JUnit Mockito

This project is a solid template for Spring REST applications. It contains basic user management (POST /register, POST /login), email verification logic (GET /verify) and also it can handle avatars. I used Spring Security and JWT authenication. All implemented logic is tested with automated unit and integration tests. I used MySQL in production environment with Flyway migration control and H2 database for test purposes.

Source code is uploaded to GitHub and final application is deployed to Heroku. You can try it out with Postman based on API description which is available in the respository as well.
